# 🧠 Mental Health Prediction System

Machine Learning web app using real-world dataset to predict mental health risk.

## 🚀 Features
- Real Kaggle dataset (1000+ rows)
- ML pipeline (scaling + encoding)
- Multiple models comparison
- Streamlit UI + login

## ▶️ Run
pip install -r requirements.txt  
cd src  
python model.py  
cd ../app  
streamlit run app.py
